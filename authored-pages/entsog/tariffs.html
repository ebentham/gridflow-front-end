Overview

What this dataset is.

tariffs holds the full published tariff prices per operator-point-direction — unit prices, capacity types, currency and validity windows. It is the canonical price reference for European gas transmission capacity. It shares the schema-absent operational shape; endpoint-specific fields cover the connection narrative, balancing-zone pair, product window and FX reference. Each row is one tariff line for an operator-point-direction over a validity window.

It is sourced from the ENTSO-G Transparency Platform at /api/v1/tariffsFulls with a default countryKey=UK filter and timeZone=UCT. The raw JSON lands in bronze and is written to silver via GenericEntsogJsonTransformer (subclassed as TariffsTransformer), with no Pydantic class — columns are derived dynamically.

As the capacity-price reference, it is the price side of the price-volume pair with firm_available, pairs with cmp_auction_premiums to derive total transmission cost at congested points, and is the line-item companion to tariff_simulations. Schema and API verified against the gridflow connector source and the 2026-05-08 vault Bronze sample; ENTSO-G is a public, no-authentication API.

-- UK annual-firm tariff trajectory by operator (last 5 gas years)
SELECT EXTRACT(year FROM valid_from) AS gas_year,
       operator, direction_key, value,
       operator_currency
FROM read_parquet('data/silver/entsog/tariffs/**/*.parquet')
WHERE tariff_capacity_type = 'Firm'
  AND country_code = 'GB'
  AND valid_from >= '2020-10-01'
ORDER BY gas_year, operator, direction_key;
import polars as pl

df = pl.read_parquet("data/silver/entsog/tariffs/**/*.parquet")
# Current-gas-year firm tariffs at GB operators
current = (
    df.filter(pl.col("valid_from") <= pl.lit("2026-05-20"))
      .filter(pl.col("valid_to")   >  pl.lit("2026-05-20"))
      .filter(pl.col("tariff_capacity_type") == "Firm")
      .sort("value", descending=True)
)
print(current.select(["operator", "direction_key", "value", "unit"]).head())
Caveats

Things to know before using this.

These are specific to tariffs. Also read the ENTSO-G-wide caveats on the vendor page.

01
Default countryKey=UK filter

The connector passes countryKey=UK by default. For full EU tariffs, override the param. connectors/entsog/endpoints.py L186

02
Unit is EUR/(kWh/h)/year for annual products

Tariff prices are denominated per unit-of-booked-capacity per validity-window. To compute annual cost: value × kWh/h × (year-fraction-of-validity). vault Bronze sample; tariff convention

03
timeZone=UCT (vendor typo, intentional)

Connector sends UCT not UTC; sending UTC returns HTTP 400. connectors/entsog/endpoints.py L17 · ENTSOG_TIMEZONE

04
Empty windows return HTTP 404, not 200

ENTSO-G's empty convention is HTTP 404 + {"message":"No result found"}. Connector short-circuits as empty-bronze. connectors/entsog/client.py L24-28, L109-115

05
operator_point_direction is vendor-lowercase here

E.g. "ro-tso-0001itp-00153exit" (all-lowercase) rather than the capitalised form used in the CMP family. Don't rely on a single casing. vault Bronze sample L73
