Overview

What this dataset is.

cmp_auction_premiums holds the price outcomes of capacity auctions — the reserve price, the final cleared price, and the premium paid above reserve — per operator-point-direction. It is the canonical price signal for congested EU gas interconnection capacity: a non-zero auction_premium means demand exceeded the reserve and bidders competed up the price. Each row is one auction's price outcome for a capacity-validity window.

It is sourced from the ENTSO-G Transparency Platform at /api/v1/cmpAuctions, queried over a date range with timeZone=UCT and periodType=day. The raw JSON lands in bronze and is written to silver via GenericEntsogJsonTransformer (subclassed as CmpAuctionPremiumsTransformer), with no Pydantic class — columns are derived dynamically. The endpoint-specific numeric fields are auctionPremium, clearedPrice and reservePrice.

As the market-clearing price record for scarce capacity, it pairs with cmp_unsuccessful_requests and cmp_unavailable_firm_capacity to connect scarcity to price, and with the tariffs family to interpret the capacity-product economics. Schema and API verified against the gridflow connector source and the 2026-05-08 vault Bronze sample; ENTSO-G is a public, no-authentication API.

-- BBL exit annual-auction premium time series
SELECT EXTRACT(year FROM auction_from) AS auction_year,
       point_key, direction_key,
       AVG(reserve_price)  AS avg_reserve,
       AVG(cleared_price)  AS avg_cleared,
       AVG(auction_premium) AS avg_premium
FROM read_parquet('data/silver/entsog/cmp_auction_premiums/**/*.parquet')
WHERE point_key = 'ITP-00207'
  AND direction_key IN ('exit', 'Exit')
GROUP BY 1, 2, 3
ORDER BY 1;
import polars as pl

df = pl.read_parquet("data/silver/entsog/cmp_auction_premiums/**/*.parquet")
# Premium ratio: how much above reserve did the auction clear?
prem = (
    df.with_columns((pl.col("cleared_price") / pl.col("reserve_price")).alias("premium_ratio"))
      .filter(pl.col("premium_ratio") > 1.5)
      .sort("premium_ratio", descending=True)
)
print(prem.head(15))
Caveats

Things to know before using this.

These are specific to cmp_auction_premiums. Also read the ENTSO-G-wide caveats on the vendor page.

01
direction_key casing — check both forms

CMP-family endpoints sometimes return capitalised values; cross-endpoint joins should lowercase both sides. vault Known Issues; CMP-family convention

02
Reserve = cleared means no scarcity

When auction_premium ≈ 0 and cleared_price = reserve_price, the auction was uncongested (multiple eligible bidders below the cap, or a single bidder at reserve). domain knowledge / CAM auction mechanics

03
timeZone=UCT (vendor typo, intentional)

Connector sends UCT not UTC; sending UTC returns HTTP 400. connectors/entsog/endpoints.py L17 · ENTSOG_TIMEZONE

04
Empty windows return HTTP 404, not 200

ENTSO-G's empty convention is HTTP 404 + {"message":"No result found"}. Connector short-circuits as empty-bronze. connectors/entsog/client.py L24-28, L109-115

05
Prices are EUR/(kWh/h/year)-style products

The vendor unit is "EUR" but the implicit capacity-product reference (yearly/quarterly/monthly/daily kWh/h) determines the actual EUR-per-energy interpretation. Read productType if present (in the tariffs/tariff_simulations family). ENTSO-G data dictionary
