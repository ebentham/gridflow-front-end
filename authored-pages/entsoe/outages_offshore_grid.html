Overview

What this dataset is.

outages_offshore_grid holds unavailability of offshore grid infrastructure in MW per host bidding zone — offshore wind cable and substation outages with asset identification. Each row carries an asset_mrid and asset_name identifying the specific offshore asset offline (e.g. "Dogger Bank A export cable", "Hornsea 2 substation"). Unlike outages_transmission (zone-pair scope), A79 uses single-zone scope — the offshore grid is attached to a host zone.

It is sourced from the ENTSO-E Transparency Platform at /api?documentType=A79&BiddingZone_Domain={EIC}, queried per zone. Unlike most outage surfaces, A79 has no BusinessType filter pinned by the connector — only DocStatus and mRID are in optional_params. The raw XML lands in bronze, is validated against EntsoeOutagesOffshoreGrid (schemas/entsoe.py L506-528), and written to silver via OutagesOffshoreGridTransformer in the H7 outages family.

Offshore grid outages are particularly important for short-term wind forecast error attribution — a substation or cable failure directly explains a B18 (Wind Offshore) generation drop that the meteorological forecast cannot anticipate. Pair against wind_solar_forecast to measure forecast skill, and against actual_generation (B18 rows) to identify the output drop. Schema verified against gridflow source on 2026-05-20; the live API requires extended ENTSO-E registration.

Caveats

Things to know before using this.

These are specific to outages_offshore_grid. Also read the ENTSO-E-wide caveats on the vendor page.

01
Optional parameters per connector — no BusinessType pinned

A79 has only DocStatus and mRID in optional_params — no BusinessType filter is pinned. This returns all outage types in one call. endpoints.py L75-81

02
asset_mrid not unit_mrid

A79 uses asset_mrid (cable / substation) — same shape as outages_transmission. Don't confuse with the unit_mrid used by generation outage surfaces. schemas/entsoe.py L511-512

03
H7 transformer family

Uses shared _H7OutageTransformer base. Single-zone domain (not zone-pair). silver/entsoe/outages_h7.py L160-179

04
outage_type is Optional

For this dataset only, outage_type defaults to "" (not required). Some A79 publications don't carry a businessType. schemas/entsoe.py L513

05
Entitlement-blocked on default API key

Live API returns HTTP 401 for the unregistered gridflow default key. .planning/reconciliation/entsoe/25-outages-offshore-grid-http-401.md
