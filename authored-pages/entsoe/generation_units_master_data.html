Overview

What this dataset is.

generation_units_master_data is the EIC reference registry of production and generation units under DocumentType A95 — the lookup that turns an opaque unit_mrid into a human-readable name, a PSR type, and a commissioning date. Unlike the time-series surfaces, it is reference data: keyed by (area_code, unit_mrid) with no timestamp_utc. Each row holds a unit's unit_name, production_type, and implementation_datetime_utc (the date the unit started operation).

It is sourced from the ENTSO-E Transparency Platform at /api?documentType=A95&BusinessType=B11&BiddingZone_Domain={EIC}, with the connector pinning BusinessType=B11 (production unit) and using Implementation_DateAndOrTime as the date param rather than the usual periodStart/periodEnd. The raw XML lands in bronze, is validated against EntsoeGenerationUnitsMasterData (schemas/entsoe.py L258-274), and written to silver via GenerationUnitsMasterDataTransformer, which filters out rows with an empty unit_mrid.

It is the join key behind every unit-level analysis: attach it to actual_generation_units, installed_capacity_units, or outages_generation on unit_mrid to recover plant names and types. Its GB analogue is Elexon bmunits_reference, since A95 is empty for GB. Schema verified against gridflow source on 2026-05-20; the live API requires extended ENTSO-E registration.

Caveats

Things to know before using this.

These are specific to generation_units_master_data. Also read the ENTSO-E-wide caveats on the vendor page.

01
No timestamp_utc — reference table

This dataset is keyed by (area_code, unit_mrid); it doesn't carry a time-series timestamp. implementation_datetime_utc is the unit's commissioning date, not the row's publication time. schemas/entsoe.py L258-267

02
BusinessType B11 is required

Connector pins BusinessType=B11 (Production unit). Other BusinessType values (B14 generation unit, etc.) return different surfaces — out of scope for this dataset. endpoints.py L115-122

03
date_param uses Implementation_DateAndOrTime

Unlike most ENTSO-E endpoints which use periodStart/periodEnd, A95 uses Implementation_DateAndOrTime as the date param. endpoints.py L121

04
Empty unit_mrid rows filtered

Transformer drops rows where unit_mrid == "" (silver/entsoe/generation_units_master_data.py L64). gridflow source

05
Entitlement-blocked on default API key

Live API returns HTTP 401 for the unregistered gridflow default key. Schema verified from gridflow source. .planning/reconciliation/entsoe/15-generation-units-master-data-http-401.md
