Overview

What this dataset is.

regional_wales is current carbon intensity and generation mix for Wales (regionid 17) — the canonical live country-level snapshot for Wales's gas-dominated grid. Each period carries an ahead-of-period forecast_gco2_kwh and a post-period actual_gco2_kwh, with a categorical intensity_index band of very low / low / moderate / high / very high. Values sit on the GB 30-minute settlement grid in UTC. Each row represents one region's reading for one half-hour (and, where present, one fuel).

It is sourced from the NESO Carbon Intensity API at api.carbonintensity.org.uk/regional/wales, queried for the named GB nation, validated against RegionalIntensity; the silver transformer (RegionalWalesTransformer) is generated dynamically at import by register_neso_transformers(). The API is a public JSON-REST endpoint — no authentication, just an Accept: application/json header — and the transformer walks the period-keyed regions[] envelope and flattens each region's intensity / generationmix into rows before writing the silver parquet partition.

As the spatial GB carbon signal, it drives region-aware load shifting and back-tests, and pairs with Elexon fuelhh and Open-Meteo weather for locational marginal-emission work. Verified against vendor docs: 2026-05-08 · NESO Carbon Intensity · /regional/wales ↗

-- Wales gas share trend over 30 days
SELECT timestamp_utc::date AS day, avg(generation_percentage) AS mean_gas_pct
FROM read_parquet('data/silver/neso/regional_wales/**/*.parquet')
WHERE fuel = 'gas'
  AND timestamp_utc >= current_date - INTERVAL 30 DAY
GROUP BY 1
ORDER BY 1 DESC;
