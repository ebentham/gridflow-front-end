Overview

What this dataset is.

regional_intensity_fw24h_regionid is regionid-scoped carbon-intensity forecast 24 hours ahead — the canonical day-ahead carbon outlook for one numeric-keyed GB DNO region. Each period carries an ahead-of-period forecast_gco2_kwh and a post-period actual_gco2_kwh, with a categorical intensity_index band of very low / low / moderate / high / very high. Values sit on the GB 30-minute settlement grid in UTC. Each row represents one region's reading for one half-hour (and, where present, one fuel).

It is sourced from the NESO Carbon Intensity API at api.carbonintensity.org.uk/regional/intensity/{from}/fw24h/regionid/{regionid}, queried per NESO region id, validated against RegionalIntensity; the silver transformer (RegionalIntensityFw24HRegionidTransformer) is generated dynamically at import by register_neso_transformers(). The API is a public JSON-REST endpoint — no authentication, just an Accept: application/json header — and the transformer walks the period-keyed regions[] envelope and flattens each region's intensity / generationmix into rows before writing the silver parquet partition.

As the spatial GB carbon signal, it drives region-aware load shifting and back-tests, and pairs with Elexon fuelhh and Open-Meteo weather for locational marginal-emission work. Verified against vendor docs: 2026-05-08 · NESO Carbon Intensity · /regional/intensity/{from}/fw24h/regionid/{regionid} ↗

-- Multi-region day-ahead comparison: cleanest forecast SP per region
SELECT regionid, shortname,
       min(forecast_gco2_kwh) AS min_g,
       arg_min(timestamp_utc, forecast_gco2_kwh) AS cleanest_sp
FROM read_parquet('data/silver/neso/regional_intensity_fw24h_regionid/**/*.parquet')
WHERE fuel = ''
GROUP BY 1, 2
ORDER BY min_g;
