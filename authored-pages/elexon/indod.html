Overview

What this dataset is.

indod is the daily-aggregate GB demand outturn — daily-rolled version of indo for peak-day comparisons, monthly totals, and year-on-year demand trends. One row per settlement_date; no settlement_period.

Gridflow fetches it from /datasets/INDOD using the publishDateTimeFrom / publishDateTimeTo pattern. The raw JSON lands in bronze and is written to the silver parquet partition via INDODTransformer — no Pydantic class; cadence is genuinely daily-only.

Refreshed daily with 1 day publication lag. Verified against vendor docs on 2026-05-08.

-- Monthly demand totals over the last 12 months
SELECT date_trunc('month', settlement_date) AS month,
       SUM(initial_demand_outturn_mw) AS monthly_mwh,
       AVG(initial_demand_outturn_mw) AS avg_daily_mwh
FROM read_parquet('data/silver/elexon/indod/**/*.parquet')
WHERE settlement_date >= current_date - INTERVAL 12 MONTH
GROUP BY 1 ORDER BY 1;
