Overview

What this dataset is.

tsdfd is the daily-published 2-to-14-day-ahead GB transmission demand forecast — the medium-horizon companion to tsdf for forward transmission margin, transmission planning, and embedded-generation projections. Daily-aggregate granularity, no settlement-period concept.

Gridflow fetches it from /datasets/TSDFD using the publishDateTimeFrom / publishDateTimeTo pattern. The raw JSON lands in bronze and is written to the silver parquet partition via TSDFDTransformer — keyed on forecast_date rather than settlement-date because the horizon is daily, not per-period.

Refreshed daily with 12-day horizon and 0 publication lag. Verified against vendor docs on 2026-05-08.

-- TSDFD vs NDFD spread (implied embedded contribution at the medium horizon)
SELECT t.forecast_date,
       n.national_demand_mw  AS national_forecast_mw,
       t.forecast_demand_mw  AS transmission_forecast_mw,
       (n.national_demand_mw - t.forecast_demand_mw) AS implied_embedded_mw
FROM read_parquet('data/silver/elexon/tsdfd/**/*.parquet') t
JOIN read_parquet('data/silver/elexon/ndfd/**/*.parquet') n
  ON t.forecast_date = n.settlement_date
WHERE t.forecast_date >= current_date
  AND t.forecast_date <= current_date + INTERVAL 14 DAY
ORDER BY t.forecast_date;
