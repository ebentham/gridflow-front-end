Overview

What this dataset is.

netbsad is the per-period netted GB BSAD adjustments — buy and sell price/volume aggregates that feed cash-out price formation. It is the canonical reference for SBP/SSP attribution, BSC settlement reconciliation, and DISBSAD totals validation.

Gridflow fetches it from /datasets/NETBSAD using the publishDateTimeFrom / publishDateTimeTo pattern. The raw JSON lands in bronze and is written to the silver parquet partition via NETBSADTransformer — no Pydantic class; the live API now returns 8 finer-grained fields where the transformer maps only 4, so fresh bronze will produce silver with the 4 mapped columns null until remapping.

Refreshed daily with 1 day publication lag. Verified against vendor docs on 2026-05-08.

-- Daily NETBSAD-derived buy-side price contribution
SELECT settlement_date,
       SUM(net_buy_price_adjustment) AS total_buy_adjustment_gbp_mwh,
       AVG(net_buy_volume_adjustment) AS avg_buy_volume_mwh
FROM read_parquet('data/silver/elexon/netbsad/**/*.parquet')
WHERE settlement_date >= current_date - INTERVAL 30 DAY
  AND net_buy_price_adjustment IS NOT NULL
GROUP BY 1 ORDER BY 1;
