Overview

UOU2T14D provides declared generation availability for every active BM Unit across every settlement period in the 2–14 day-ahead window. Each row carries the BM Unit identifier, forecast date, and usable output in MW. This is the unit-level dataset from which FOU2T14D (fleet-level totals) is derived.

The volume is substantial: a single 4-hour query window covering the 2-week horizon returns hundreds of thousands of rows. The vendor enforces a hard 4-hour maximum query window — requests wider than 4 hours return HTTP 400. The connector enforces max_chunk_hours=4 and chunks accordingly.

The silver schema maps forecastDate → settlement_date (same pattern as FOU2T14D). No Pydantic model exists — UOU2T14DTransformer enforces the 7-column shape via Polars casting.

SELECT settlement_date, settlement_period, bm_unit_id,
       output_usable_mw
FROM read_parquet('data/silver/elexon/uou2t14d/**/*.parquet')
WHERE settlement_date = '2026-05-08'
  AND settlement_period = 1
ORDER BY bm_unit_id;
import polars as pl

df = pl.read_parquet("data/silver/elexon/uou2t14d/**/*.parquet")
# Fleet total per settlement period
fleet = (
    df.group_by(["settlement_date", "settlement_period"])
    .agg(pl.col("output_usable_mw").sum().alias("fleet_total_mw"))
    .sort(["settlement_date", "settlement_period"])
)
print(fleet)

Caveats

01
Vendor hard cap — 4-hour maximum query window

The Elexon API returns HTTP 400 for query windows wider than 4 hours. The connector enforces max_chunk_hours=4. A full 2-week backfill requires 84 sequential 4-hour chunks — plan for significant ingestion time and storage.

02
Very high row count — filter aggressively

A single 4-hour window across the 2-week horizon returns hundreds of thousands of rows (one per active BM unit × settlement period). Always filter by settlement_date, settlement_period, or bm_unit_id before loading into memory.

03
forecastDate maps to settlement_date — same pattern as FOU2T14D

The API response carries forecastDate which the transformer maps to settlement_date. This is consistent with FOU2T14D and the rest of the 2-14 day availability family.
