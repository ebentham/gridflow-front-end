-- Year-on-year demand comparison
SELECT
  date_part('month', settlement_date) AS month,
  date_part('year', settlement_date) AS year,
  round(avg(initial_demand_outturn_mw), 0) AS avg_daily_demand
FROM silver.indod
WHERE date_part('year', settlement_date) >= 2022
GROUP BY 1, 2 ORDER BY 2, 1;

Caveats

Things to know.

Dataset-specific notes. Also see Elexon-wide caveats.

01
Scale is daily total, not instantaneous MW

The value for a typical day is ~600,000–700,000 — this is the sum of 48 half-hourly periods. Do not compare directly to INDO values (which are ~25,000–35,000 MW per period).

02
Only 30 rows per month

INDOD is very small. For most analyses you can load it entirely into memory without filtering.
