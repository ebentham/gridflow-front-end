-- Volume-weighted composite MID price per period
SELECT
  settlement_date, settlement_period,
  round(
    sum(market_index_price * market_index_volume) / nullif(sum(market_index_volume), 0)
  , 2) AS composite_price
FROM silver.mid
WHERE settlement_date = '2026-05-01'
GROUP BY 1, 2 ORDER BY 2;
import pandas as pd

mid = pd.read_parquet("data/silver/elexon/mid/",
    filters=[("settlement_date", ">=", "2026-04-01")])
# composite VWAP across both providers
vwap = (mid.groupby(['settlement_date','settlement_period'])
        .apply(lambda x: (x.market_index_price * x.market_index_volume).sum() / x.market_index_volume.sum()))
Caveats

Things to know.

Dataset-specific notes. Also see Elexon-wide caveats.

01
from/to params — not settlementDate

MID requires ?from=ISO&to=ISO. Using ?settlementDate= returns a 404. The connector handles this but note it if querying the API manually.

02
Two rows per period (one per provider)

Each settlement period has two rows — one for APXMIDP and one for N2EXMIDP. Filter or aggregate before joining to single-row-per-period datasets.

03
market_index_volume can be null

If a provider has no trades in a period, market_index_volume may be null. Apply COALESCE or fill before computing VWAP.
