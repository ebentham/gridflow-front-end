-- Forward margin: total available capacity minus demand forecast
SELECT
  f.settlement_date,
  sum(f.output_usable_mw) AS total_available_mw,
  avg(n.national_demand_mw) AS forecast_demand_mw,
  sum(f.output_usable_mw) - avg(n.national_demand_mw) AS margin_mw
FROM silver.fou2t14d f
JOIN silver.ndfd n USING (settlement_date, settlement_period)
GROUP BY 1 ORDER BY 1;

Caveats

Things to know.

Dataset-specific notes. Also see Elexon-wide caveats.

01
Silver layer keeps only latest forecast per target

FOU2T14D is published daily. The silver layer deduplicates on (settlement_date, settlement_period, fuel_type) — only the most recent forecast per target is retained.

02
Wind availability is a forecast, not a guarantee

WIND output_usable_mw is a forecast of available wind, not metered capacity. Actual output depends on weather — use fuelhh WIND for actuals.
