Architecture

Bronze · silver · gold.

Raw API responses land in bronze. Cleaned, typed, deduped tables live in silver. Joined, analysis-ready views in gold.

Everything is Hive-partitioned Parquet, queryable from DuckDB or the Python client. Backfills are idempotent. Schema changes are caught at parse time by Pydantic v2 contracts.

Models

Pipeline → signal.

Data sources stabilising · models next

The warehouse exists to support quantitative work. Three models are planned once all data sources are stable.

Generation Planned

Power Stack & Merit Order

Derive a half-hourly GB merit order from fuelhh and bmunits_reference, combined with fuel and carbon costs. Reconcile modelled dispatch against Elexon system prices to identify pricing anomalies.

Forecasting Planned

Wind & Solar Forecast Error

Decompose NESO wind forecast error by horizon, weather regime, and season using windfor vs agws. Benchmark a naïve persistence model and explore weather-conditioned correction.

Fundamentals Planned

Storage Trajectory vs. Norm

EU gas storage fill rate vs. 5-year seasonal band, with weather-adjusted demand overlay from Open-Meteo. Identify deviation from norm as a fundamental signal for gas price direction.

Every model on this site is a function of the catalogue. Inputs are linked. Code is on GitHub. Caveats are stated.

Query it

SQL or Python — pick one.

All examples use the same canonical silver schema.
-- GB residual demand last 30 days, hourly
SELECT
  date_trunc('hour', settlement_dt) AS hour,
  sum(case when fuel_type = 'WIND'  then mw end) AS wind_mw,
  sum(case when fuel_type = 'NPSHYD' then mw end) AS hydro_mw,
  sum(mw) AS total_mw
FROM silver.fuelhh
WHERE settlement_dt >= now() - interval '30 days'
GROUP BY 1
ORDER BY 1;
from gridflow import GridflowClient

with GridflowClient() as gf:
    df = gf.get_fuel_generation(
        start="2026-04-01",
        end="2026-04-29",
    )
    # pivot to one column per fuel type
    pivot = df.pivot(
        values="generation_mw",
        index="timestamp_utc",
        columns="fuel_type",
    )
import pandas as pd

df = pd.read_parquet(

  columns=["timestamp_utc", "fuel_type", "generation_mw"],
)
df = df[df.timestamp_utc >= pd.Timestamp.now() - pd.Timedelta("30d")]
